# Changelog

All notable changes to LittleCMS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-02

### Added
- Initial release of LittleCMS
- Git-powered CMS functionality
- GitHub file embedding feature (migrated from GitShow)
- Cloudflare Workers deployment support
- CLI tool for initialization
- TypeScript support throughout
- Modern React admin UI foundation (structure ready)
- Markdown rendering utilities
- HTML escaping utilities

### Changed
- Migrated from GitShow to LittleCMS branding
- Converted worker code to TypeScript
- Restructured project for npm package distribution

### Removed
- Legacy GitShow-specific references
